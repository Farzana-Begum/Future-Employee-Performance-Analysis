This project analyzes employee performance, satisfaction, experience, and attrition trends to help organizations make informed HR decisions. The dataset was cleaned and prepared using SQL, and an interactive performance dashboard was created using Power BI.
## Project Overview
The goal of this project is to understand key workforce metrics such as performance ratings, job satisfaction, work-life balance, and attrition. Insights from the analysis support HR teams in improving employee engagement, productivity, and retention strategies.
## Project Objectives
- Analyze employee performance, satisfaction, and experience.
- Identify key factors influencing attrition and retention.
- Evaluate department-wise workforce trends and engagement.
- Understand links between job satisfaction, salary hikes, and performance.
- Provide HR with actionable insights through a Power BI dashboard.
